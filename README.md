# ZerOS 1.0

ZerOS 1.0 — учебный DOS-подобный проект на C с установщиком, командной строкой, упрощенной файловой системой ZFM и дополнительными утилитами.

## Что реализовано

- Установка при запуске (`y/n`) с предупреждением об удалении файлов.
- Выбор директории-диска для установки.
- Форматирование выбранной директории и создание структуры ZFM:
  - `ZFM/users`
  - `ZFM/docs`
  - `ZFM/system`
- Создание пользователя и опционального пароля.
- Консоль ZerOS с зелёным фоном и белым текстом в баннере.
- ASCII-логотип и подсказка `Введи help чтобы узнать команды`.
- Базовые команды shell и файловой системы.
- Простые игры (`guess`, `coin`).
- Мини IDE-команда для генерации шаблона C# приложения.

## Команды

- `help`
- `clear`, `clean`
- `version`
- `echo <text>`
- `prompt <name>`
- `time`, `date`, `month`, `season`
- `hw`, `mem`
- `ls`, `pwd`, `cd <dir>`, `mkdir <dir>`, `touch <file>`, `cat <file>`, `write <file> <text>`, `rm <file>`, `rmdir <dir>`
- `notepad <file>`
- `game guess`, `game coin`
- `ide`
- `exit`

## Сборка

```bash
make
```

## Запуск

```bash
make run
```

## Сборка ISO

```bash
make iso
```

> Для ISO нужен `xorriso` или `genisoimage`.
> ISO в этом проекте — установочный образ-симуляция с бинарником `zeros`.
